# Recursos del curso

Esta carpeta contiene material reutilizable por los alumnos.

Para los enlaces externos principales, lee tambien [../recursos.md](../recursos.md).

## Enlaces importantes

### Cromatogramas de trabajo

[Cromatogramas GC-MS del curso](https://drive.google.com/drive/folders/1lRc_glpeQNt-PXHD4sLunmMkT75XJl2G?usp=share_link)

Aqui estan los cromatogramas con los que se trabajara durante el curso. Si los descargas localmente, mantenlos fuera de Git, por ejemplo en `data/raw/`.

### Referencias para NotebookLM

[Referencias para NotebookLM](https://drive.google.com/drive/folders/1hHobZbc4fEVaL7YF3qN5BhDrxhqBE6wB?usp=share_link)

Estas referencias se cargaran en NotebookLM. Durante el curso se usara NotebookLM como RAG para consultas sobre papers y documentos, conectado al flujo de trabajo con Antigravity IDE.

## Subcarpetas

- `datos-sinteticos/`: datasets publicos generados para practicar.
- `prompts/`: plantillas de prompts usadas en la clase.
- `lecturas/`: lista de papers y fuentes recomendadas.

No subir aqui datos reales de instrumento, datos personales, resultados confidenciales ni PDFs con restricciones de distribucion.
