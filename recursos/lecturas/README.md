# Lecturas recomendadas

Pendiente de completar con el instructor.

Formato recomendado para cada lectura:

```text
- Autor, ano. Titulo. Revista. DOI/enlace.
  Uso en el curso: metodo, biomarcador, limitacion o contexto biologico.
```

No subir PDFs con copyright salvo que la licencia permita redistribucion. Para GitHub, es mejor enlazar DOI, PubMed, publisher o preprint.
