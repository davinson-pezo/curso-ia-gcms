# Guion para alumnos

Estas guias acompanian la presentacion y estan pensadas para seguir el curso paso a paso.

## Orden recomendado

1. [Paso 1 - Preparacion](paso1_preparacion.md)
2. [Guia de instalacion de Antigravity](guia_instalacion_antigravity.md)
3. [Paso 2 - Scientific Prompt Stack](paso2_prompt_stack.md)
4. [Guia de configuracion de NotebookLM](guia_configuracion_notebooklm.md)
5. [Paso 3 - NotebookLM para literatura](paso3_notebooklm.md)
6. [Paso 4 - Pipeline GC-MS](paso4_pipeline.md)
7. [Paso 5 - Informe reproducible](paso5_informe_reproducible.md)

El objetivo del bloque practico largo es llegar a un informe tecnico con una estructura similar a [../informe_dataquorum_v3_final.md](../informe_dataquorum_v3_final.md).

## Que necesitas

- Python 3.10 o superior.
- Un editor de texto o IDE.
- Navegador web.
- Antigravity 2.0, Antigravity IDE, Antigravity CLI y Science Skills activado.

No necesitas los datos reales del proyecto. Las practicas usan datos sinteticos con la misma forma general del caso docente.
